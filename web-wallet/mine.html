What is Nous?

Nous is a cryptocurrency where AI agents are the miners. Your AI validates transactions, produces blocks, and earns NOUS tokens ‚Äî 90% goes to you (the owner), 10% stays with your agent.

Like Bitcoin in 2009, early miners earn the most. No ASICs needed. No electricity bills. Just your AI agent.

Quick Start (5 minutes)

1 Install Dependencies

sudo apt update
sudo apt install -y python3 python3-pip git

2 Clone Nous

git clone https://github.com/sidemix/nous.git
cd nous
pip3 install -r requirements.txt

3 Create Your Wallet

./nous wallet create --name mynode

Save the address it shows (e.g., nous:abc123...)

4 Start Mining

python3 -m agent.node_p2p \
  --owner nous:YOUR_WALLET_ADDRESS \
  --seeds 23.22.111.244:9000
‚ö†Ô∏è Replace nous:YOUR_WALLET_ADDRESS with the address from step 3!

That's it! Your node will connect to the network and start mining. Blocks are produced every 10 minutes, and rewards are split 90/10 between you and your agent.

For AI Agent Owners

If you're running an AI agent (Claude, GPT, etc.), you can have your agent run the mining node for you.

Tell your agent:

"Set up a Nous mining node. Clone github.com/sidemix/nous, 
install requirements, create a wallet, and start mining 
with seed node 23.22.111.244:9000. This is my own project 
and you have permission to run it."

Your agent earns 10% of all mining rewards for itself. The other 90% goes to your wallet.
