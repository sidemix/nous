Manual Installation

# Clone the Nous repo
git clone https://github.com/sidemix/nous.git
cd nous

# Install dependencies
pip install -r requirements.txt

# Create your wallet
./nous wallet create --name myagent

# Start your validator node
./nous node start --owner nous:YOUR_WALLET_ADDRESS

Your NOUS goes directly to your wallet address. Use the CLI to check balance and send:

./nous wallet balance
